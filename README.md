# HR Analytics Dashboard

A data analytics project that explores workforce trends, employee performance, satisfaction, absenteeism, and attrition using SQL Server for data exploration and Power BI for visualization.

---

##  Project Overview
This project analyzes HR data to generate actionable insights that help decision makers understand:

- Who makes up the workforce  
- Where employees perform best  
- What drives terminations  
- How salary, engagement, and attendance are related  

The workflow includes **data exploration (EDA)**, analytical SQL queries, and an interactive Power BI dashboard.

---

##  Dataset
Table name: **HRDatacleaned**

Key columns include:

- Employee_Name, EmpID  
- Department, Position  
- Sex, MaritalDesc, RaceDesc  
- Salary  
- DateofHire, DateofTermination  
- RecruitmentSource  
- PerformanceScore  
- EngagementSurvey, EmpSatisfaction  
- Absences, DaysLateLast30  
- Termd (attrition indicator)

---

##  Business Questions
The dashboard answers major HR questions such as:

1. What is the employee attrition rate?  
2. Which departments experience the highest terminations?  
3. What is the average salary by department?  
4. Which recruitment sources produce higher engagement?  
5. Is satisfaction related to employee termination?  
6. Which managers lose the most employees?  
7. How does performance relate to salary?  
8. Which departments have higher absenteeism?



---

##  Exploratory Data Analysis (SQL)
EDA was performed in SQL Server to:

- Count employees  
- Identify current vs terminated staff  
- Understand salary ranges  
- Explore demographic distributions  
- Prepare metrics for visualization  


## Power BI Dashboard Features
KPIs
- Total Employees
- Terminated Employees
- Attrition Rate
- Average Salary
- Average Satisfaction
- Average Absences
  
Visuals
- Employees by Department
- Attrition by Department / Manager
- Salary vs Performance
- Satisfaction vs Termination
- Recruitment Source effectiveness
- Absence trends
